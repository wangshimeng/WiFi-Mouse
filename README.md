# WiFi Mouse 🖱️
The ultimate productivity tool that turns your mobile device into a powerful wireless input system for PC, Mac, and Linux.

### ⌨️ Main Capabilities
* **Fully Functional Mouse:** Left/Right click, middle scroll, and drag-and-drop support.
* **Remote Keyboard:** Type on your computer using your phone’s soft keyboard (supports all languages).
* **Multi-Touch Gestures:** Two-finger scroll, pinch-to-zoom, and three-finger swipe for window management.
* **Presentation Mode:** Control PPT/Keynote slides from a distance.

### ⚙️ How it Works
1. Install the WiFi Mouse App on your Android.
2. Ensure your phone and PC are on the same Wi-Fi network.
3. Auto-discover or manually connect via IP for instant, low-latency control.
